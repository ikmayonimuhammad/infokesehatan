# 🏥 Kanal Informasi Kesehatan

Aplikasi web berbasis PHP Native hasil gabut + nyoba AI yang dimana fungsi aplikasi menyediakan informasi dan artikel seputar kesehatan, gaya hidup sehat, pola makan, serta tips menjaga kebugaran tubuh.

---

## 📌 Fitur Utama
-  Artikel kesehatan
-  Menu admin
-  mode malam
-  Tampilan sederhana dan mudah digunakan
-  page dinamis

---

## 🛠️ Teknologi yang Digunakan
- PHP (Native)
- HTML
- CSS (opsional)
- MySQL (jika menggunakan database)
