🏥 Kanal Informasi Kesehatan

Ini adalah aplikasi web sederhana berbasis PHP Native yang dibuat karena gabut + sekalian eksperimen pakai AI 😄
Fungsinya? Buat nyediain berbagai info dan artikel seputar kesehatan, mulai dari gaya hidup sehat, pola makan, sampai tips biar tubuh tetap fit.

📌 Fitur Utama
📰 Artikel kesehatan yang update dan informatif
🔐 Panel admin buat kelola konten
🌙 Mode malam (biar nyaman di mata)
✨ Tampilan simpel, ringan, dan gampang dipakai
🔄 Halaman dinamis
🛠️ Teknologi yang Dipakai
PHP (Native)
HTML
CSS (opsional, kalau mau dikembangin lagi)
MySQL (opsional, kalau pakai database)
🎁 Catatan

Project ini gratis dan bebas dipakai sesuai kebutuhan.
Namanya juga proyek gabut, jadi silakan dimodif, dikembangin, atau dijadiin bahan belajar 👍
